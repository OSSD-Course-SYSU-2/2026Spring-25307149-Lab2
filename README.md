项目介绍
本项目基于 HarmonyOS NEXT 6.0.2 ArkTS 开发 2D 太空射击休闲小游戏，为操作系统系统开发课程实验作业。游戏采用 Canvas 绘制游戏画面，自定义悬浮虚拟摇杆 + 右侧 2×2 技能操作面板，完成飞船移动、射击、道具拾取、生命值计分、敌机生成等完整游戏逻辑。
核心功能
游戏核心玩法
飞船跟随虚拟摇杆全方位移动，边界碰撞限制；
射击、多段攻击、全屏清屏、子弹流转四类技能；
随机生成钻石、敌机等道具 / 障碍物，碰撞判定；
生命值、得分实时 UI 展示，游戏结束判定逻辑。
多端自适应适配
动态获取设备屏幕尺寸与系统安全区，自动避让顶部状态栏、底部手势条；
区分手机竖屏 / 平板横屏布局，摇杆、技能按钮按屏幕比例自动缩放；
Touch 触摸坐标归一化处理，解决不同设备分辨率触控偏移、按键消失问题；
全页面采用相对百分比布局，无固定硬编码宽高数值，手机、模拟器、平板一键部署运行。
交互优化
左侧圆形虚拟摇杆，松手自动回弹，低误触设计；
右侧紧凑圆形技能按键组，悬浮于游戏画布上层，不遮挡游戏视野。
